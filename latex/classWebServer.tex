\doxysubsection{Detailed Description}
Simulated web server that processes a single \doxylink{structRequest}{Request} at a time. 

A \doxylink{classWebServer}{Web\+Server} either has an active \doxylink{structRequest}{Request} or is idle. Each call to \doxylink{classWebServer_aa06a71feffcbf17bf58070c54a829a39}{process\+Tick()} advances the current \doxylink{structRequest}{Request} by one clock cycle.
