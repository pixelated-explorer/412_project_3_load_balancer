\chapter{Rayan Mohammed\+: CSCE 412 Project 3 – Load Balancer}
\hypertarget{index}{}\label{index}\index{Rayan Mohammed: CSCE 412 Project 3 – Load Balancer@{Rayan Mohammed: CSCE 412 Project 3 – Load Balancer}}
This project implements a discrete-\/event simulation of a web load balancer. The system manages a queue of client Requests and a pool of \doxylink{classWebServer}{Web\+Server} instances, dynamically scaling the number of servers based on queue length.

Key features\+:
\begin{DoxyItemize}
\item Random generation of requests with IP addresses, processing times, and job types (such as streaming vs. processing).
\item A \doxylink{classLoadBalancer}{Load\+Balancer} class that assigns work to servers and keeps track of time through \textquotesingle{}ticks\textquotesingle{} or clock cycles.
\item Dynamic scaling of servers to keep the queue between 50 and 80 times the number of servers runing, with configurations for wait cycles between each scaling action.
\item An IP range blocker that acts as a simple firewall / DoS protection.
\item Console color output to highlight scaling events and status, plus logging to a text file including configuration and summary statistics.
\end{DoxyItemize}

The documentation pages for the \doxylink{structRequest}{Request}, \doxylink{classWebServer}{Web\+Server}, and \doxylink{classLoadBalancer}{Load\+Balancer} classes describe how these components interact to simulate a real time load balancer.
