\doxysubsection{Detailed Description}
Entry point for the load balancer simulation. 

This program configures a \doxylink{classLoadBalancer}{Load\+Balancer} either interactively or via a simple configuration file, runs the simulation for the requested number of clock cycles, and writes both console output and a log file.

Program entry point. 

Steps\+:
\begin{DoxyItemize}
\item Ask the user for a log file name.
\item Ask whether to enable verbose per-\/server logging.
\item Optionally load configuration parameters from a file in the {\ttfamily configs/} directory.
\item Prompt for any parameters not supplied by the configuration file.
\item Create a \doxylink{classLoadBalancer}{Load\+Balancer}, seed the initial request queue, and run the simulation for the requested number of ticks.
\item Print and log a summary of results.
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
0 on success, non-\/zero on failure to open the log file.
