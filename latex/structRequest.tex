\doxysubsection{Detailed Description}
Represents a single web request in the load balancer simulation. 

A \doxylink{structRequest}{Request} contains\+:
\begin{DoxyItemize}
\item source IP address (ip\+In)
\item destination IP address (ip\+Out)
\item processing time (needed\+Time) in clock cycles
\item job type (job\+Type) such as "{}stream"{} or "{}process"{} 
\end{DoxyItemize}
