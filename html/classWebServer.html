Detailed Description

Simulated web server that processes a single Request at a time.

A WebServer either has an active Request or is idle. Each call to processTick() advances the current Request by one clock cycle.

bool WebServer::processTick ( )

Process one clock tick of work.

If the server is working, this decrements timeLeft by one. When timeLeft reaches zero, the request is considered complete and the server becomes idle again.

Returns
true if a request finished on this tick, otherwise false.
